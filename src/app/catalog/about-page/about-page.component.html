Views Control Component

Управление видами - инструмент управления отображением списка товаров. Представляет собой блок с элементами управления для выбора способа сортировки списка, количества выводимых на странице товаров и способа отображения товаров на странице. Предоставляет пользователю возможность выбрать удобный для его целей способ представления информации.

Задачи:

  1. выводить блок управления способом отображения товаров на странице;
  2. генерировать элемент управления для выбора способа сортировки;
  3. генерировать элемент управления для выбора количества отображаемыж на странице товарных позиций;
  4. генерировать элемент управления для выбора способа отображения на странице товарных позиций;
  5. передавать в родительский компонент данные о выбраном способе представления информации.

Pages Control Component

Управление страницами - инструмент управления постраничным выводом информации. Представляет собой навигационный блок с ссылками или кнопками, которые используются для перехода на другие страницы представления.

Задачи:

  1. выводить блок постраничной навигации по каталогу, если список товаров представлен на нескольких страницах;
  2. генерировать список ссылок для постраничной навигации;
  3. управлять активностью ссылок для постраничной навигации;
  4. передавать в родительский компонент информацию о номере выбраной страницы.

Реализация:

  1. компонент принимает от родительского массив pages объектов типа Pagination;
  2. шаблон компонента отображает разметку блока в том случае, если массив pages содержит более одного элемента;
  3. кнопки постраничной навигации с номерами генерируются в результате перебора элементов массива pages;
  4. кнопки Предыдущая и Следущая представлены в шаблоне компонента;
  5. при клике на кнопке постраничной навигации срабатывает функция обратного вызова onClick, в котрую передается объект события и индекс элемента;
  6. функция onClick отменяет для кнопки действие по умолчанию и передает в родительский компонент информацию о выбраной странице;
  7. состояние кнопок постраничной навигации с номерами управляется классом active;
  8. состояние кнопок Предыдущая и Следущая управляется атрибутом disabled через свойства компонента prevButtonDisabledStatus и nextButtonDisabledStatus соответсвенно.
